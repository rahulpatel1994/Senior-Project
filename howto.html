Condense the Web!

An extension that provides a user text based analysis.

Steps for Summarization Use:

  1. Enter requested amount of sentences.
  2. Wait for information to be populated.
  3. Re-enter if needed.

Hashtag Use:

  1. 10 tags are populated by default. 10 additional are created with button press.

Keywords:

  • Keywords is a gloassary of words generated from the page. Each keyword(s) sends the user to a wikipedia page based on the keyword selected.

Citations:

  • Citations will try and generate a MLA formatted cite. It will also provide the author and the publish date
